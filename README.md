# PES_Grid
A smart electrical grid prototype running on a decentralised server(Blockchain).  
  
# Dependencies
<ul>
  <li>Node js</li>
  <li>Truffle</li>
  <li>Solidity</li>
</ul>  
  
# How to run 
Run the command 'npm install' to install the dependencies.  
To run the server enter the command 'npm run dev'.
